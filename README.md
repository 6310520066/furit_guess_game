# furit_guess_game

### To run this game

#### open 3 terminals

##### 1<sup>st</sup> terminal for compile and start server
```
gcc -o server server.c -lpthread
gcc -o client client.c
./server
```

##### 2<sup>nd</sup> terminal for 1st client
```
./client
```

##### 3<sup>rd</sup> terminal for 2<sup>nd</sup> client
```
./client
```
